# EchoSpeak App Icon 设计方案

## 设计理念
EchoSpeak 是一个语音回声应用,图标应该体现:
- 🎤 **语音/麦克风** - 核心功能是录音
- 🔊 **声波/回声** - 播放回声效果
- 🎨 **现代科技感** - 简洁、现代的设计风格

## 图标设计方案

### 方案一: 麦克风 + 声波
```
┌─────────────────────┐
│                     │
│      🎤             │
│    ))) )))          │  麦克风图标,周围有同心圆声波
│  )))     )))        │  表示声音的传播和回声
│    ))) )))          │
│                     │
└─────────────────────┘
```

**配色方案:**
- 主色: 深蓝色 (#2563EB) → 青色 (#06B6D4) 渐变
- 背景: 白色或浅灰色
- 声波: 半透明的青色圆环

### 方案二: 字母 E + 声波
```
┌─────────────────────┐
│                     │
│       ╔═══          │
│       ║             │  字母 "E" (Echo) 
│       ╠═══          │  右侧有声波纹理
│       ║      )))    │  简洁现代
│       ╚═══   )))    │
│                     │
└─────────────────────┘
```

**配色方案:**
- 主色: 渐变蓝色
- 强调色: 亮青色用于声波

### 方案三: 圆形声波图案(推荐)
```
┌─────────────────────┐
│                     │
│    ╭───────╮        │
│   ╱  ╭───╮  ╲       │  同心圆声波
│  │  ╱ 🎤 ╲  │      │  中心是麦克风
│   ╲  ╰───╯  ╱       │  外围是扩散的声波
│    ╰───────╯        │
│                     │
└─────────────────────┘
```

**配色方案:**
- 背景: 深蓝色渐变 (#1E40AF → #2563EB)
- 声波圆环: 从内到外透明度递减的青色
- 麦克风: 白色或亮青色

## 技术规格

### Android Adaptive Icon
- **尺寸**: 512x512px (上传到 Play Store)
- **安全区域**: 中心 66% 区域(避免被裁剪)
- **格式**: PNG (带透明背景) 或 XML Vector

### iOS App Icon
- **尺寸**: 1024x1024px
- **圆角**: 系统自动处理
- **格式**: PNG (不带透明背景)

## 实现建议

1. **使用 Figma/Sketch 设计**
   - 创建 1024x1024 画布
   - 使用矢量图形
   - 导出多种尺寸

2. **使用 Android Studio Image Asset**
   - 导入 512x512 图标
   - 自动生成所有需要的尺寸

3. **在线工具**
   - https://icon.kitchen/ - 快速生成 Android Adaptive Icon
   - https://appicon.co/ - 生成所有平台图标
